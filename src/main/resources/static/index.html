API--随机从url库中获取随即一张图片

https://api.ayaka.icu/img/url

参数：

//非必要参数, 不写默认为 bili=1
bili=1 : 横图
bili=2 : 竖图
bili=3 : 类似正方形 (较少)

示例1：

https://api.ayaka.icu/img/url

示例2：

https://api.ayaka.icu/img/url?bili=2

API--随机从url库中获取随机图片 JSON 字符串

https://api.ayaka.icu/img/url/json

参数：

//非必要参数, 不写默认为 bili=1
bili=1 : 横图
bili=2 : 竖图
bili=3 : 类似正方形 (较少)

访问API

https://api.ayaka.icu/img/url/json

JSON数据示例:

{
    "size" : "0.58MB" #图片大小
    "type" : "jpg" #图片大小
    "url" : "https://ayaka-icu-oss.oss-cn-beijing.aliyuncs.com/img/cat/cat02.png" #图片地址
    "width" : "2000" #图片宽度
    "height" : "1125" #图片高度
}

API--获取给定诺干の诺干URL-随机一个重定向

https://api.ayaka.icu/img/tool/urls?urls=xxx@@xxx

参数：

urls是必要参数 

urls=路径1@@路径2@@更多 

多个路径以 @@ 相分割

示例1：

https://api.ayaka.icu//img/tool/urls?urls=http://img.ayaka.icu/i/2023/03/25/641e74f1770ad.jpg@@http://img.ayaka.icu/i/2023/03/25/641e80711778b.jpg

示例2：

不仅适用于图片连接，其他连接依然适用
https://api.ayaka.icu/img/tool/urls?urls=https://blog.ayaka.icu@@https://cloud.ayaka.icu

注意：

File库中的图片并不多，File库图片源于该网站部署的服务器，站长可以放入自己的图片




API--获取给定诺干の诺干URL-随机一个响应体返回图片

https://api.ayaka.icu/img/tool/urls.io?urls=xxx@@xxx

参数：

urls是必要参数 

urls=路径1@@路径2@@更多 

多个路径以 @@ 相分割

示例:

https://api.ayaka.icu/img/tool/urls.io?urls=http://img.ayaka.icu/i/2023/03/25/641e74f1770ad.jpg@@http://img.ayaka.icu/i/2023/03/25/641e80711778b.jpg

注意：

- 非图片链接无法使用！
- 性能 /img/tool/urls?urls=xxx@@xxx 差
- 非必要请勿使用！




API-- 获取数字图片-图片计数器

https://api.ayaka.icu/img/count?num=666

参数：

//必要参数: num 指定要计数的数字

num=666  

//非必要参数: max 指定要计数格式，前位未满补 0，不写默认为 6 位
//当 num > max  时，跟随 num 大小： max = num 

max=6

示例1：

http://api.ayaka.icu/img/count?num=789

效果1：

示例2：

http://api.ayaka.icu/img/count?num=123456&max=8

效果2：

示例3：

https://api.ayaka.icu/img/count?num=87654321

效果3：

注意：

允许最大位数为 11 位，后续可能扩大......




API--获取给定诺干个Icon名称，返回这些图标，转发

https://api.ayaka.icu/icons?i=icon1,icon2,icon3

参数：

//i:必要 , theme:非必要 , size:非必要

- i : 给定的icon参数
  - 多个以英文 , 相隔开
  - 更多图标参数请查看: https://github.com/tandpfun/skill-icons#icons-list 

- theme : 主题色
  - 1 为黑暗色 默认
  - 2 为明亮色

- size : 每行最多多少个图片，超过自动分行
  - 默认为 15

示例1：

http://api.ayaka.icu/icons?i=java,idea,nginx,docker,redis,spring,mysql,github,linux,lua,md

效果1：

示例2：

http://api.ayaka.icu/icons?i=java,idea,nginx,docker,redis,spring,mysql,github,linux,lua,md&theme=2

效果2：

示例3：

http://api.ayaka.icu/icons?i=java,idea,nginx,docker,redis,spring,mysql,github,linux,lua&size=5

效果3：

示例4：

http://api.ayaka.icu/icons?i=java,idea,nginx,docker,redis,spring,mysql,github,linux,lua&theme=2&size=5
